# Arm Selection Research

## Recommendation

Delay full arm integration until the mobile base can navigate reliably, unless manipulation is the core objective.

## Required arm data

- Payload.
- Reach.
- DOF.
- Repeatability.
- Weight.
- Power.
- ROS 2 driver support.
- MoveIt 2 config availability.
- Mounting pattern.
- E-stop/fault behavior.
