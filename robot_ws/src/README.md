# ROS 2 Packages

Package prefix: `buddy_`.

- `buddy_description`: robot model and frames.
- `buddy_bringup`: launch and configuration.
- `buddy_base`: base hardware interface/bridge.
- `buddy_firmware_interfaces`: MCU message/protocol contract.
- `buddy_navigation`: Nav2, SLAM, localization.
- `buddy_perception`: camera/LiDAR/perception pipelines.
- `buddy_simulation`: Gazebo worlds and simulation launch.
- `buddy_manipulation`: future arm/MoveIt 2 placeholder.
- `buddy_diagnostics`: health aggregation.
- `buddy_mission`: behavior/task logic.
- `buddy_tests`: tests and validation scenarios.
- `buddy_operator`: operator dashboards/UI notes.
