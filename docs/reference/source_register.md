# Source Register

Initial repository scaffold was derived from the ROS 2 Autonomous Robot Design Master Roadmap PDF included in this folder and from current official documentation checked during repo generation.

Key topics to recheck before final hardware decisions:

- ROS 2 Jazzy/Lyrical supported platforms.
- Isaac ROS supported platforms and ROS distro support.
- Nav2 documentation and behavior tree examples.
- ROS 2 drivers for selected LiDAR, RGB-D camera, IMU, motor drivers, and robot arm.
